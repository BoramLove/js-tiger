

### this 关键字

this 指的就是当前对象

```
class Person {
  constructor(name) {

  }
  sayName() {
    console.log('my name is' + name);
  }
}
let peter = new Person('peter');
peter.sayName();
```

此时如果直接看 peter.sayName() 会发现 `name` 的值为空。

解决这个问题就可以使用 this 关键字。

```
class Person {
  constructor(name) {
    this.name = name;
  }
  sayName() {
    console.log('my name is' + this.name);
  }
}
let peter = new Person('peter');
peter.sayName();
```

这样 sayName() 函数中就可以拿到 this.name 的值了。
